::numeros_enteros::¿Cuál es la definición de Números Enteros? {
=Conjunto que incluye números positivos, negativos y el cero. #¡Correcto! Los números enteros (ℤ) incluyen todos los números positivos (+1, +2, +3...), todos los números negativos (-1, -2, -3...) y el cero (0). Este conjunto es fundamental en matemáticas.
~Conjunto que incluye solo números positivos y el cero únicamente. #Incorrecto. Esta definición excluye los números negativos, que son parte esencial del conjunto de números enteros. Los enteros incluyen positivos, negativos y cero.
~Conjunto que incluye números fraccionarios, decimales y el cero. #Incorrecto. Esta definición corresponde más a los números racionales. Los enteros son solo números "completos" sin partes fraccionarias.
}

// question: 2  name: valor_absoluto
::valor_absoluto::¿Cuál es la definición de Valor Absoluto? {
=Distancia de un número al cero en la recta numérica. #¡Correcto! El valor absoluto siempre es positivo porque representa distancia. Por ejemplo, |−5| = 5 y |5| = 5, ambos están a 5 unidades del cero.
~Distancia de un número al uno en la recta numérica. #Incorrecto. El valor absoluto mide la distancia al cero, no al uno. Es el punto de referencia fundamental en la recta numérica.
~Distancia de un número al infinito en la recta numérica. #Incorrecto. El infinito no es un punto específico en la recta. El valor absoluto siempre se mide desde el cero como punto de referencia.
}

// question: 3  name: recta_numerica
::recta_numerica::¿Cuál es la definición de Recta Numérica? {
=Línea donde cada punto representa un número entero único ordenadamente. #¡Correcto! La recta numérica es una representación visual donde cada punto corresponde a exactamente un número entero, organizados de menor a mayor de izquierda a derecha.
~Línea donde cada punto representa un número decimal único ordenadamente. #Incorrecto. Aunque los decimales también pueden representarse en una recta, en el contexto de números enteros nos enfocamos en la representación de enteros.
~Línea donde cada punto representa un número fraccionario único ordenadamente. #Incorrecto. Las fracciones se estudian en el módulo de números racionales. Aquí nos enfocamos en la representación de números enteros.
}

// question: 4  name: opuesto
::opuesto::¿Cuál es la definición de Opuesto de un número entero? {
=Número con igual valor absoluto pero signo contrario. #¡Correcto! El opuesto de un número tiene la misma distancia al cero pero en dirección contraria. Por ejemplo, el opuesto de +7 es -7, y viceversa.
~Número con diferente valor absoluto pero mismo signo. #Incorrecto. Los números opuestos deben tener el mismo valor absoluto. Si tienen diferente valor absoluto, no son opuestos entre sí.
~Número con igual valor absoluto y mismo signo. #Incorrecto. Si tienen el mismo signo, son números iguales, no opuestos. Los opuestos requieren signos contrarios.
}

// question: 5  name: conjunto_z
::conjunto_z::¿Cuál es la definición de Conjunto ℤ? {
=Símbolo que representa al conjunto de los números enteros. #¡Correcto! La letra ℤ (zeta) proviene de la palabra alemana "Zahlen" que significa números. Es la notación matemática estándar para los números enteros.
~Símbolo que representa al conjunto de los números racionales. #Incorrecto. Los números racionales se representan con ℚ (de "quotient"). ℤ específicamente representa solo los números enteros.
~Símbolo que representa al conjunto de los números naturales. #Incorrecto. Los números naturales se representan con ℕ. ℤ incluye además los números negativos y el cero.
}

// question: 6  name: suma_enteros
::suma_enteros::¿Cuál es la definición de Suma de Enteros? {
=Operación que combina números enteros siguiendo reglas específicas de signos. #¡Correcto! La suma de enteros sigue reglas específicas: signos iguales se suman, signos diferentes se restan, y el resultado toma el signo del número con mayor valor absoluto.
~Operación que combina enteros positivos y fraccionarios correctamente. #Incorrecto. Esta definición mezcla enteros con fraccionarios, que pertenecen a diferentes conjuntos numéricos. La suma de enteros solo involucra números enteros.
~Operación que combina enteros negativos y decimales correctamente. #Incorrecto. Los decimales no son números enteros. La suma de enteros se limita a operar solo con números del conjunto ℤ.
}

// question: 7  name: numero_primo
::numero_primo::¿Cuál es la definición de Número Primo? {
=Número natural mayor que uno con exactamente dos divisores distintos. #¡Correcto! Un número primo tiene exactamente dos divisores: el 1 y él mismo. Por ejemplo, 7 es primo porque solo es divisible entre 1 y 7. Los primeros primos son: 2, 3, 5, 7, 11, 13...
~Número natural mayor que uno con exactamente tres divisores distintos. #Incorrecto. Un número con tres divisores sería un número compuesto (como 9 = 1×3×3). Los primos tienen exactamente dos divisores, no tres.
~Número natural mayor que dos con exactamente dos divisores distintos. #Incorrecto. Esta definición excluiría al número 2, que es el único número primo par y el más pequeño de todos los primos.
}

// question: 8  name: mcd
::mcd::¿Cuál es la definición de MCD (Máximo Común Divisor)? {
=Mayor número entero que divide exactamente a dos o más números dados. #¡Correcto! El MCD (Máximo Común Divisor) es el número más grande que divide a todos los números dados sin dejar residuo. Por ejemplo, MCD(12,18) = 6.
~Menor número entero que divide exactamente a dos o más números dados. #Incorrecto. El menor divisor común de cualquier conjunto de números siempre es 1. El MCD busca el MAYOR divisor común, no el menor.
~Mayor número entero que es múltiplo de dos o más números dados. #Incorrecto. Esta definición se refiere a múltiplos, no a divisores. El MCD se basa en encontrar divisores, no múltiplos.
}

// question: 9  name: mcm
::mcm::¿Cuál es la definición de MCM (Mínimo Común Múltiplo)? {
=Menor número entero positivo que es múltiplo común de dos o más números dados. #¡Correcto! El MCM (Mínimo Común Múltiplo) es el número más pequeño que es múltiplo de todos los números dados. Por ejemplo, MCM(4,6) = 12.
~Mayor número entero positivo que es múltiplo común de dos o más números dados. #Incorrecto. No existe un mayor múltiplo común porque los múltiplos se extienden al infinito. El MCM busca el MENOR múltiplo común.
~Menor número entero positivo que es divisor común de dos o más números dados. #Incorrecto. Esta definición corresponde al concepto de divisor común, que se relaciona con el MCD. El MCM trabaja con múltiplos, no divisores.
}

// question: 10  name: factorizacion
::factorizacion::¿Cuál es la definición de Factorización Prima? {
=Descomposición de un número en producto únicamente de factores primos. #¡Correcto! La factorización prima expresa cualquier número como multiplicación de números primos. Por ejemplo, 12 = 2² × 3. Esta descomposición es única para cada número.
~Descomposición de un número en suma únicamente de factores primos. #Incorrecto. La factorización usa multiplicación (producto), no suma. Por ejemplo, 12 = 2×2×3, no 2+2+3.
~Descomposición de un número en producto únicamente de factores compuestos. #Incorrecto. La factorización prima requiere específicamente números primos, no compuestos. Los factores compuestos pueden descomponerse más.
}

// question: 11  name: criterios_divisibilidad
::criterios_divisibilidad::¿Cuál es la definición de Criterios de Divisibilidad? {
=Reglas matemáticas para determinar si un número es divisible por otro sin realizar la división. #¡Correcto! Los criterios de divisibilidad son métodos rápidos para verificar divisiones sin calcular. Por ejemplo, un número es divisible entre 2 si termina en cifra par.
~Reglas matemáticas para determinar si un número es múltiplo de otro sin realizar operaciones. #Incorrecto. Los criterios se enfocan específicamente en divisibilidad, no en verificar si es múltiplo directamente.
~Reglas matemáticas para determinar si un número es factor de otro sin realizar operaciones. #Incorrecto. Los criterios verifican divisibilidad, que es equivalente a ser factor, pero la terminología correcta es divisibilidad.
}

// question: 12  name: descomposicion_prima
::descomposicion_prima::¿Cuál es la definición de Descomposición Prima? {
=Expresar un número natural como producto único de potencias de números primos. #¡Correcto! La descomposición prima es única para cada número según el Teorema Fundamental de la Aritmética. Por ejemplo, 30 = 2 × 3 × 5.
~Expresar un número natural como suma única de potencias de números primos. #Incorrecto. La descomposición prima utiliza multiplicación (producto), no suma. La suma de primos no garantiza unicidad ni utilidad matemática.
~Expresar un número natural como producto único de potencias de números compuestos. #Incorrecto. Debe usar factores primos, no compuestos. Los números compuestos pueden descomponerse más, por lo que no serían la forma "prima" final.
}

// question: 13  name: numeros_racionales
::numeros_racionales::¿Cuál es la definición de Números Racionales? {
=Números que pueden expresarse como cociente de dos enteros con denominador diferente de cero. #¡Correcto! Los números racionales (ℚ) se escriben como a/b donde a y b son enteros y b ≠ 0. Incluyen fracciones como 3/4, enteros como 5/1, y decimales finitos como 0.5 = 1/2.
~Números que pueden expresarse como cociente de dos enteros con denominador igual a cero. #Incorrecto. La división entre cero no está definida en matemáticas. El denominador siempre debe ser diferente de cero para que la fracción tenga sentido.
~Números que pueden expresarse como producto de dos enteros con denominador diferente de cero. #Incorrecto. Los racionales se definen como cociente (división), no como producto (multiplicación). El producto de enteros siempre da otro entero.
}

// question: 14  name: fracciones_equivalentes
::fracciones_equivalentes::¿Cuál es la definición de Fracciones Equivalentes? {
=Fracciones que representan la misma cantidad pero con numeradores y denominadores diferentes. #¡Correcto! Las fracciones equivalentes tienen el mismo valor pero diferentes numeradores y denominadores. Por ejemplo: 1/2 = 2/4 = 3/6 = 4/8. Se obtienen multiplicando o dividiendo por el mismo número.
~Fracciones que representan diferentes cantidades con numeradores y denominadores iguales. #Incorrecto. Si tienen numeradores y denominadores iguales, son fracciones idénticas, no equivalentes. Las equivalentes tienen diferente apariencia pero igual valor.
~Fracciones que representan la misma cantidad con numeradores y denominadores iguales. #Incorrecto. Si numeradores y denominadores son iguales, las fracciones son idénticas. Las equivalentes tienen diferente forma pero mismo valor.
}

// question: 15  name: simplificacion
::simplificacion::¿Cuál es la definición de Simplificación de fracciones? {
=Reducir una fracción dividiendo numerador y denominador por su máximo común divisor. #¡Correcto! Simplificar significa dividir numerador y denominador por su MCD para obtener la fracción más simple. Por ejemplo: 12/18 ÷ 6 = 2/3.
~Aumentar una fracción dividiendo numerador y denominador por su máximo común divisor. #Incorrecto. Simplificar significa reducir o hacer más simple, no aumentar. El proceso busca la expresión más básica de la fracción.
~Reducir una fracción multiplicando numerador y denominador por su máximo común divisor. #Incorrecto. Multiplicar por el MCD haría la fracción más compleja, no más simple. La simplificación requiere división, no multiplicación.
}

// question: 16  name: fraccion_propia
::fraccion_propia::¿Cuál es la definición de Fracción Propia? {
=Fracción donde el numerador es menor que el denominador. #¡Correcto! Las fracciones propias representan valores menores que 1. Ejemplos: 3/4, 5/8, 2/7. Su valor decimal siempre es menor que 1.0.
~Fracción donde el numerador es mayor que el denominador. #Incorrecto. Esta definición corresponde a fracciones impropias. Cuando el numerador es mayor, el valor es mayor que 1.
~Fracción donde el numerador es igual al denominador. #Incorrecto. Cuando numerador y denominador son iguales, la fracción vale exactamente 1, no es propia ni impropia en sentido estricto.
}

// question: 17  name: fraccion_impropia
::fraccion_impropia::¿Cuál es la definición de Fracción Impropia? {
=Fracción donde el numerador es mayor que el denominador. #¡Correcto! Las fracciones impropias representan valores mayores que 1. Ejemplos: 7/4, 9/5, 11/3. Pueden convertirse a números mixtos como 7/4 = 1¾.
~Fracción donde el numerador es menor que el denominador. #Incorrecto. Esta definición corresponde a fracciones propias. Cuando el numerador es menor, el valor es menor que 1.
~Fracción donde el numerador es igual al denominador. #Incorrecto. Cuando son iguales, la fracción vale exactamente 1. Para ser impropia, el numerador debe ser estrictamente mayor.
}

// question: 18  name: numero_decimal
::numero_decimal::¿Cuál es la definición de Número Decimal? {
=Representación numérica que utiliza punto decimal para expresar fracciones. #¡Correcto! Los números decimales expresan fracciones usando punto decimal. Por ejemplo: 0.75 = 3/4, 0.5 = 1/2. Pueden ser finitos (0.25) o infinitos periódicos (0.333...).
~Representación numérica que utiliza coma decimal para expresar fracciones. #Incorrecto. Aunque algunos países usan coma decimal, la definición estándar y más universal utiliza punto decimal como separador.
~Representación numérica que utiliza punto decimal para expresar únicamente números enteros. #Incorrecto. Los números decimales representan específicamente fracciones, no enteros. Los enteros no necesitan punto decimal para su representación.
}

// question: 19  name: razon
::razon::¿Cuál es la definición de Razón? {
=Comparación cuantitativa entre dos cantidades mediante división o cociente. #¡Correcto! Una razón compara dos cantidades dividiéndolas. Se expresa como a:b o a/b. Por ejemplo, si hay 8 niños y 12 niñas, la razón es 8:12 o 2:3 simplificada.
~Comparación cuantitativa entre dos cantidades mediante suma o resta. #Incorrecto. Las razones se basan en división, no en suma. La suma combina cantidades, mientras que la razón las compara proporcionalmente.
~Comparación cuantitativa entre tres cantidades mediante división o cociente. #Incorrecto. Una razón básica compara exactamente dos cantidades. Aunque existen razones complejas, la definición fundamental involucra dos elementos.
}

// question: 20  name: proporcion
::proporcion::¿Cuál es la definición de Proporción? {
=Igualdad matemática entre dos razones que relaciona cuatro términos. #¡Correcto! Una proporción establece que dos razones son iguales: a/b = c/d. Los términos a, b, c, d se llaman proporcionales. Por ejemplo: 2/3 = 8/12.
~Desigualdad matemática entre dos razones que relaciona cuatro términos. #Incorrecto. Una proporción requiere igualdad, no desigualdad. Si las razones no son iguales, no forman una proporción válida.
~Igualdad matemática entre tres razones que relaciona seis términos. #Incorrecto. Una proporción básica compara exactamente dos razones con cuatro términos. Tres razones formarían una proporción compuesta más compleja.
}

// question: 21  name: regla_tres_directa
::regla_tres_directa::¿Cuál es la definición de Regla de Tres Directa? {
=Método para resolver problemas donde las magnitudes aumentan o disminuyen en la misma proporción. #¡Correcto! En proporcionalidad directa, cuando una magnitud aumenta, la otra también aumenta en la misma proporción. Ejemplo: más trabajadores, más trabajo realizado.
~Método para resolver problemas donde las magnitudes cambian en proporciones independientes. #Incorrecto. En la regla de tres directa, las magnitudes cambian en la misma dirección (proporcionalmente), no de manera independiente.
~Método para resolver problemas donde las magnitudes aumentan y disminuyen de forma inversa. #Incorrecto. Esta definición describe la regla de tres inversa, no directa. En la directa, ambas magnitudes van en la misma dirección.
}

// question: 22  name: regla_tres_inversa
::regla_tres_inversa::¿Cuál es la definición de Regla de Tres Inversa? {
=Método para resolver problemas donde una magnitud aumenta mientras la otra disminuye proporcionalmente. #¡Correcto! En proporcionalidad inversa, cuando una magnitud aumenta, la otra disminuye en proporción inversa. Ejemplo: más velocidad, menos tiempo para recorrer la misma distancia.
~Método para resolver problemas donde ambas magnitudes aumentan proporcionalmente. #Incorrecto. Esta definición describe la regla de tres directa, no inversa. En la inversa, las magnitudes van en direcciones opuestas.
~Método para resolver problemas donde ambas magnitudes disminuyen proporcionalmente. #Incorrecto. Si ambas disminuyen juntas, sería proporcionalidad directa. En la inversa, una aumenta mientras la otra disminuye.
}

// question: 23  name: porcentaje
::porcentaje::¿Cuál es la definición de Porcentaje? {
=Forma de expresar una proporción como fracción de cien partes. #¡Correcto! El porcentaje (%) representa partes de 100. Por ejemplo, 25% = 25/100 = 0.25. Es útil para comparar proporciones de diferentes totales.
~Forma de expresar una proporción como fracción de diez partes. #Incorrecto. Esto sería un decimal con denominador 10. El porcentaje específicamente usa 100 como base de comparación.
~Forma de expresar una proporción como fracción de mil partes. #Incorrecto. Esto sería un permil (‰). El porcentaje usa específicamente 100 como denominador estándar.
}

// question: 24  name: proporcionalidad_directa
::proporcionalidad_directa::¿Cuál es la definición de Proporcionalidad Directa? {
=Relación entre dos variables donde al aumentar una, la otra también aumenta en la misma razón. #¡Correcto! En proporcionalidad directa, ambas variables cambian en la misma dirección manteniendo una razón constante. Se expresa como y = kx, donde k es la constante de proporcionalidad.
~Relación entre dos variables donde al aumentar una, la otra disminuye en proporción inversa. #Incorrecto. Esta definición describe proporcionalidad inversa, no directa. En la directa, ambas cantidades van en la misma dirección.
~Relación entre dos variables donde ambas disminuyen sin mantener proporción constante. #Incorrecto. En proporcionalidad directa se debe mantener una razón constante, y puede incluir tanto aumento como disminución, pero siempre en la misma dirección.
}

// question: 25  name: factor_conversion
::factor_conversion::¿Cuál es la definición de Factor de Conversión? {
=Fracción equivalente a la unidad que permite cambiar unidades sin alterar la cantidad. #¡Correcto! Un factor de conversión es una fracción igual a 1 que permite cambiar unidades. Por ejemplo: 1 metro/100 centímetros = 1. Al multiplicar, cancela unidades y mantiene el valor real.
~Fracción equivalente a cero que permite cambiar unidades sin alterar la cantidad. #Incorrecto. Un factor igual a cero anularía completamente la cantidad. Los factores de conversión deben valer 1 para preservar el valor original.
~Fracción equivalente a la unidad que permite cambiar unidades alterando la cantidad proporcionalmente. #Incorrecto. El propósito del factor de conversión es cambiar solo las unidades, nunca alterar la cantidad real que se está midiendo.
}

// question: 26  name: sistema_internacional
::sistema_internacional::¿Cuál es la definición de Sistema Internacional de Unidades? {
=Sistema métrico decimal estándar mundial basado en siete unidades fundamentales. #¡Correcto! El Sistema Internacional (SI) usa unidades base: metro (longitud), kilogramo (masa), segundo (tiempo), ampere (corriente), kelvin (temperatura), mol (cantidad) y candela (intensidad luminosa).
~Sistema métrico decimal estándar mundial basado en unidades imperiales fundamentales. #Incorrecto. Las unidades imperiales (pulgada, libra) no pertenecen al Sistema Internacional. El SI usa metro y kilogramo como unidades base.
~Sistema métrico decimal estándar nacional basado en siete unidades fundamentales. #Incorrecto. El Sistema Internacional es mundial, no nacional. Su adopción es global para facilitar la comunicación científica y comercial internacional.
}

// question: 27  name: sistema_imperial
::sistema_imperial::¿Cuál es la definición de Sistema Imperial de Unidades? {
=Sistema de medidas tradicional usado principalmente en Estados Unidos y algunos países anglosajones. #¡Correcto! El Sistema Imperial usa unidades como pulgadas, pies, yardas, millas (longitud), libras, onzas (masa), y galones, cuartos (volumen). Gradualmente está siendo reemplazado por el SI.
~Sistema de medidas tradicional usado principalmente en Francia y países europeos. #Incorrecto. Francia usa el Sistema Internacional (métrico), no el Imperial. Francia fue pionera en el desarrollo del sistema métrico.
~Sistema de medidas tradicional usado principalmente en todos los países del mundo. #Incorrecto. La mayoría de países usan el Sistema Internacional. Estados Unidos es uno de los pocos países que aún usa principalmente el Sistema Imperial.
}

// question: 28  name: unidades_longitud
::unidades_longitud::¿Qué magnitud física se mide con unidades como metros, centímetros, kilómetros o pulgadas? {
=Longitud o distancia entre dos puntos. #¡Correcto! Estas unidades sirven para medir la longitud, es decir, la distancia, altura o anchura de un objeto. En SI: milímetro, centímetro, metro, kilómetro. En Imperial: pulgada, pie, yarda, milla.
~Masa o cantidad de materia de un objeto. #Incorrecto. La masa se mide en gramos, kilogramos, libras, onzas, no en metros o pulgadas.
~Tiempo o duración de un evento. #Incorrecto. El tiempo se mide en segundos, minutos, horas, días, no en metros o pulgadas.
}

// question: 29  name: conversion_sistemas
::conversion_sistemas::¿Cómo se define la Conversión entre Sistemas de Unidades? {
=Proceso matemático de cambiar medidas del sistema métrico al imperial o viceversa. #¡Correcto! La conversión entre sistemas requiere factores específicos. Ejemplos: 1 metro = 3.28 pies, 1 kilogramo = 2.2 libras, 1 litro = 0.26 galones aproximadamente.
~Proceso matemático de cambiar medidas del sistema métrico al sistema decimal. #Incorrecto. El sistema métrico ya usa notación decimal. No existe conversión entre "métrico" y "decimal" porque son compatibles.
~Proceso matemático de cambiar medidas entre diferentes unidades del mismo sistema. #Incorrecto. Esto sería conversión dentro del mismo sistema. La conversión entre sistemas se refiere específicamente a cambiar de Imperial a Métrico o viceversa.

// question: 30  name: unidades_masa

=Medidas estándar para cuantificar la cantidad de materia que contiene un objeto. #¡Correcto! Las unidades de masa cuantifican materia. En SI: miligramo, gramo, kilogramo, tonelada. En Imperial: onza, libra, ton. Masa es diferente de peso (fuerza gravitacional).
~Medidas estándar para cuantificar la cantidad de energía que contiene un objeto. #Incorrecto. La energía se mide en julios (SI) o calorías, no en gramos o libras. Masa y energía son propiedades físicas diferentes.
~Medidas estándar para cuantificar el tiempo que dura un proceso en un objeto. #Incorrecto. El tiempo se mide en segundos, minutos, horas. Los objetos físicos no "contienen" tiempo; el tiempo es una dimensión independiente.
}

// question: 31  name: potenciacion
::potenciacion::¿Cuál es la definición de Potenciación? {
=Operación matemática que representa la multiplicación repetida de un mismo número. #¡Correcto! La potenciación expresa multiplicación repetida: 2³ = 2×2×2 = 8. Se compone de base (número que se multiplica) y exponente (veces que se repite).
~Operación matemática que representa la suma repetida de un mismo número. #Incorrecto. La suma repetida es multiplicación, no potenciación. Por ejemplo: 2+2+2 = 2×3 = 6, pero 2³ = 8.
~Operación matemática que representa la división repetida de un mismo número. #Incorrecto. La división repetida no es potenciación. Las potencias se basan en multiplicación, no en división.
}

// question: 32  name: radicacion
::radicacion::¿Cuál es la definición de Radicación? {
=Operación matemática inversa de la potenciación que encuentra la base original. #¡Correcto! La radicación deshace la potenciación. Si 3² = 9, entonces √9 = 3. El símbolo √ se llama radical y el número bajo él es el radicando.
~Operación matemática inversa de la suma que encuentra el número original. #Incorrecto. La operación inversa de la suma es la resta, no la radicación. La radicación es específicamente inversa de la potenciación.
~Operación matemática inversa de la multiplicación que encuentra el factor original. #Incorrecto. La operación inversa de la multiplicación es la división. La radicación invierte específicamente la potenciación.
}

// question: 33  name: exponente
::exponente::¿Cuál es la definición de Exponente? {
=Número que indica cuántas veces se multiplica la base por sí misma. #¡Correcto! En 5⁴, el exponente 4 indica que 5 se multiplica por sí mismo 4 veces: 5×5×5×5 = 625. También se llama potencia o índice.
~Número que indica cuántas veces se suma la base por sí misma. #Incorrecto. El exponente controla multiplicación repetida, no suma repetida. La suma repetida es concepto de multiplicación simple.
~Número que indica cuántas veces se divide la base por sí misma. #Incorrecto. El exponente no se relaciona con división repetida. Los exponentes negativos involucran división, pero el concepto básico es multiplicación.
}

// question: 34  name: numeros_irracionales
::numeros_irracionales::¿Cuál es la definición de Números Irracionales? {
=Números reales con expansión decimal infinita no periódica que no pueden expresarse como fracción. #¡Correcto! Los irracionales no pueden expresarse como fracción a/b. Ejemplos: π = 3.14159..., √2 = 1.41421..., e = 2.71828... Sus decimales nunca terminan ni se repiten.
~Números reales con expansión decimal finita que no pueden expresarse como fracción. #Incorrecto. Los decimales finitos (como 0.5, 0.25) son racionales porque pueden expresarse como fracciones. Los irracionales tienen decimales infinitos.
~Números reales con expansión decimal infinita periódica que no pueden expresarse como fracción. #Incorrecto. Los decimales infinitos periódicos (como 0.333... = 1/3) son racionales. Los irracionales son infinitos NO periódicos.
}

// question: 35  name: jerarquia_operaciones
::jerarquia_operaciones::¿Cuál es el propósito de la Jerarquía de Operaciones en matemáticas? {
=Establecer un orden específico y universal para resolver expresiones matemáticas complejas. #¡Correcto! El orden es: 1) Paréntesis, 2) Exponentes/Raíces, 3) Multiplicación/División (izquierda a derecha), 4) Suma/Resta (izquierda a derecha). Recordar: PEMDAS o BODMAS.
~Permitir diferentes órdenes de resolución para expresiones matemáticas complejas. #Incorrecto. El orden debe ser específico y consistente para obtener resultados correctos. Un orden variable produciría respuestas diferentes e incorrectas.
~Establecer un orden específico únicamente para resolver expresiones físicas. #Incorrecto. Aunque se aplica en física, la jerarquía de operaciones es fundamentalmente un concepto matemático que se extiende a todas las áreas.
}

// question: 36  name: raiz_cuadrada
::raiz_cuadrada::¿Cuál es la definición de Raíz Cuadrada? {
=Número que multiplicado por sí mismo produce el número original dado. #¡Correcto! La raíz cuadrada invierte el cuadrado: si x² = a, entonces √a = x. Por ejemplo: √16 = 4 porque 4² = 16. Es el caso más común de radicación.
~Número que sumado por sí mismo produce el número original dado. #Incorrecto. Si un número sumado por sí mismo da el original, sería x + x = x, lo que solo funciona para x = 0. La raíz cuadrada involucra multiplicación.
~Número que dividido por sí mismo produce el número original dado. #Incorrecto. Cualquier número dividido por sí mismo da 1 (excepto 0). Esto no define raíz cuadrada, que se basa en multiplicación.
}

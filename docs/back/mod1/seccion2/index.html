El Conjunto de los Números Enteros (Z)

Un sistema que incluye positivos, negativos y el cero

Números Naturales

Usados para contar (1, 2, 3, 4, ...)

Ejemplo:

Un estudiante de Gastronomía necesita 3 manzanas para una receta

0

El Cero

Representa ausencia o punto neutro

Ejemplo:

Un gestor turístico reporta 0 reservaciones para un tour

Números Negativos

Representan valores por debajo de cero

Ejemplo:

Un balance comercial con -$50.000 indica pérdida

Conjunto de los Números Enteros

Z = {
..., -3, -2, -1,
0,
1, 2, 3, ...
}

Cada número entero tiene un opuesto: el opuesto de 5 es -5

Aplicaciones en la vida diaria:

Temperaturas bajo cero: -5°C (cinco grados bajo cero)

Finanzas: saldo positivo +$1.500 o negativo -$800

El cero es el único número que no es ni positivo ni negativo

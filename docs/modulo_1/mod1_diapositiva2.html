El Conjunto ℤ
Los números enteros incluyen positivos, negativos y cero:
ℤ = {..., -3, -2, -1, 0, 1, 2, 3, ...}
+5
Positivo
0
Neutro
-3
Negativo
Recta Numérica
Representación visual donde cada punto es un entero único:
←|----|----|----|----|----|----|----|----|----|→
  -4   -2    0    2    4    6    8
Regla: Un número es mayor si está a la derecha.
Ejemplo: 3 > -1 porque 3 está a la derecha de -1
Valor Absoluto |x|
La distancia desde el cero, siempre positiva:
|5| = 5
5 unidades
|-3| = 3
3 unidades
|0| = 0
0 unidades

Propiedades de la Suma

Conmutativa
a/b + c/d = c/d + a/b
Asociativa
(a/b + c/d) + e/f = a/b + (c/d + e/f)
Elemento Neutro
a/b + 0 = a/b
Elemento Inverso
a/b + (-a/b) = 0

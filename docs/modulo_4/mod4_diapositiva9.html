Porcentajes y Aumentos/Descuentos

El porcentaje (%) es una forma de expresar una proporción tomando como base el número 100. La regla de tres es la herramienta fundamental para resolver problemas de porcentajes.

Tipos de Problemas

1
Hallar un % de un número:
¿Cuál es el 15% de 200?
2
Hallar qué % es un número de otro:
¿Qué % es 40 de 160?
3
Hallar un número dado su %:
Si el 20% de un número es 50, ¿cuál es el número?

Ejemplo Paso a Paso

Problema: Una tienda ofrece un descuento del 15% en un artículo que cuesta $80.000. ¿Cuánto es el descuento y cuál es el precio final?
1
Calcular el 15% de $80.000
$\frac{\$80.000 \times 15}{100} = \$12.000$
2
Precio final = $80.000 - $12.000 = $68.000

Aplicaciones por Programa

Gestión Comercial: Calcular IVA (19%) sobre ventas de $250.000
Gastronomía: Descuentos sucesivos: 20% + 10% sobre precio de $50.000
Turismo: Aumento 10% temporada alta + descuento 5% pago anticipado

Concepto de Potenciación

¿Qué es la potenciación?

La potenciación es una operación matemática que representa una multiplicación repetida de un mismo número por sí mismo.

an = a × a × a × ... × a (n factores)

Donde:

• a es la base (número que se multiplica)

• n es el exponente (cantidad de veces que se multiplica)

• an es la potencia (resultado)

Ejemplos

23 = 2×2×2 = 8
54 = 5×5×5×5 = 625
(-3)2 = (-3)×(-3) = 9
-32 = -(3×3) = -9

Aplicación en Biotecnología

El crecimiento de una población de bacterias que se duplica cada hora puede modelarse con potencias de 2.

Si inicialmente hay 1 bacteria, después de 5 horas habrá:

25 = 32 bacterias
